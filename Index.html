Voorwoord: Waarom een website

Natuurlijk is een website maken voor een stageverslag een klein beetje onorthodox, maar het leek mij perfect om een paar redenen.

1. Een website leek mij een perfect idee voor een project.

(dus daarom is dit ook gelijk project 1)

2. Ik ga volgend jaar een Software Developer opleiding doen, dus het is heel handig om het nu al een beetje te leren

3. Zelf wou ik al een hele tijd beter worden in het maken van websites, dus neem ik hier de kans om dat te doen.


Inleiding


Takenlijst

Er waren een paar taken waar ik best vaak mee bezig was, hieronder staat uitleg over die taken

TopDesk

Ik heb tijdens deze stage een heleboel geleerd over TopDesk.

Op school was ik nooit zo goed met het programma (ik had al moeite met het aanmaken van meldingen), maar nu ik het al bijna 20 week gebruik kan ik er al een heleboel meer mee.

In het begin van de stage had ik soms nog wel dat ik even was vergeten hoe bepaalde dingen moesten, maar dat heb ik gelukkig al een hele tijd niet meer

Laptops spoelen

Het spoelen ging makkelijk, doordat ik zelf ook wel vaak genoeg PC's installeer.

Alleen ging het hier een beetje anders dan dat ik thuis gewend ben, hier installeren ze de PC's namenlijk via het netwerk.

Gelukkig wist ik er al een beetje vanaf, omdat het precies hetzelfde ging op mijn oude stageplek

WI-FI Adapter resetten

Dit moest ik meestal doen op Maarsdreef. Op een of andere manier zijn daar een paar lokalen waar het internet er nooit zin in heeft. Het resetten zelf is gelukkig best makkelijk.


1. je typt internet in de zoekbalk

2. je klikt op "status"

3. je klikt op netwerk adapter resetten

Mensen helpen met problemen

Op mijn oude stageplek moest ik soms mensen helpen wanneer ze vragen hadden over hun laptop, daar vond ik het heel leuk om mensen te helpen, dus vind ik het mooi dat ik dat hier ook kan doen.
